About This Site

This website allows users to upload images and uses deep learning models to automatically detect, classify, and segment pets in the image. If no pet is detected, the system notifies the user accordingly.

The machine learning models used are

  • resnet50 – for classification
  • deeplabv3_resnet50 – for segmentation
  • YOLO – for object detection
